# Changelog

All notable changes to the HAP specification and SDKs.

## [0.4.2] - 2026-01-22

### Changed
- Clarified content hashing is optional (VA discretion via `x-content-hash`)
- Documentation language accuracy improvements
- Added SECURITY.md and CODE_OF_CONDUCT.md

### Fixed
- Removed references to non-existent hap.dev domain
- Fixed "(future)" placeholder VAs in documentation

## [0.4.1] - 2026-01-21

### Fixed
- CI workflow paths updated for renamed packages

## [0.4.0] - 2026-01-21

### Changed
- SDK packages renamed from `hap-*` to language-specific names
- Internal namespaces updated to `human-attestation` / `HumanAttestation`

## [0.3.0] - 2026-01-19

### Added
- Initial public release of HAP specification
- SDKs for TypeScript, Python, Go, Java, Ruby, PHP, C#
- VA directory structure
