# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Our Standards

We are committed to providing a welcoming and inclusive environment for all contributors.

## Reporting

If you experience or witness unacceptable behavior, please report it to: conduct@bluescroll.io

## Enforcement

Project maintainers will review and investigate all complaints and respond appropriately.

For the full text of our code of conduct, see the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
